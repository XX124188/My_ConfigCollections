```

#

# User Configurations
NAME=

REPO=

...
```

## Oh-my-zsh installation 
Run the installation script, you could change zsh theme by editing `ZSH_THEME=` in `~/.zshrc` .

* tested on Ubuntu 16.04
*   `$ ./install-zsh.sh`
